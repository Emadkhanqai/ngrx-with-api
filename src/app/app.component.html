switchMap - Cancels the current subscription/request and can cause race condition. Used for Get Requests or cancel requests like searches or autocomplete.

concatMap - Runs subscriptions/requests in order and is less performant Used for get, post or put requests when order is important.

mergeMap - Runs subscriptions/requests in parallel. Used for get, post or put requests when order is not important.

exhaustMap - Ignores all the subscriptions/requests until it is completed
